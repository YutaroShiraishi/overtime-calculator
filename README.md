# 残業代計算ツール

ブラウザで使える **無料の残業代自動計算ツール** です。
時給と労働時間を入力するだけで、残業・深夜・休日手当を即時計算できます。

👉 公開ページ
https://ユーザー名.github.io/overtime-calculator/

---

## ✨ 特徴

* 登録不要
* 完全無料
* スマホ対応
* 即時計算
* 軽量（静的サイト）

---

## 🧮 計算ルール

労働基準法の割増率に基づいて計算しています。

| 種類   | 倍率    |
| ---- | ----- |
| 通常残業 | 1.25倍 |
| 深夜残業 | 1.5倍  |
| 休日労働 | 1.35倍 |

---

## 🚀 使い方

1. 時給を入力
2. 各時間を入力
3. 「計算する」を押す

→ 合計支給額が表示されます。
